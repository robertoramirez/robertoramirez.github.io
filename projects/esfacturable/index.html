EsFacturable Software

Aplicación web basada en Windows con el objetivo de cumplir con el lineamiento del SAT para el resguardo de comprobantes fiscales digitales en México.

Modelo de operación:

El emisor o receptor de factura electrónica envía a un correo pre definido los documentos electrónicos PDF y XML. El sistema a través de tablero o dashboard permite clasificar, organizar y ordenar los documentos para una sencilla administración. A su vez nos da la opción para visualizar o descargar los documentos en formato zip. La aplicación, al momento de la recepción de los documentos electrónicos, los cuales se procesan desde la bandeja de correo electrónico, realiza las validaciones necesarias, entre las que destacan:

  • Semáforo para indicar si el estatus del comprobante sigue vigente, o bien, si fué cancelado por el emisor en el SAT.
  • Verifica que la estructura del documento XML cumpla con las especificaciones técnicas indicadas en el Anexo 20 del SAT.

Tecnología empleada:

C#, Asp.net, Javascript, Css, AngularJS, Ajax y Firebird.
